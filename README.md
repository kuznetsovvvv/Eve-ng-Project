# Сеть компании "Суперстор"
## Описание проекта
«Суперстор» расширяется и готовит новые офис и даркстор к открытию в другом городе. Компания располагается в двух локациях: главный офис, даркстор, у которого есть два помещения. 
В головном офисе IT отдел и отдел менеджмента, также сервера - DNS, Ansible, Domain Controller, Mail server, Web-server, File server. Для работы сотрудников есть внутренняя платформа: через неё управляющие складами, логисты и менеджеры видят поступления товаров, планируют маршруты доставки и управляют складскими запасами. 

Задачи:
- Назначить статические IP-адреса серверам. Всего серверных комнат две: в головном офисе и в дарксторе;
- Рассчитать адресное пространство сети 10.10.0.0/16;
- Назначить соответствующим отделам VLAN;
- NAT должен быть настроен так, чтобы все локальные VLAN могли выходить во внешнюю сеть через ближайший маршрутизатор, подключённый к провайдеру связи;
- Настроить DNAT для веб-сервера.
- Настроить прохождение трафика между внутренними сетями компании через выделенный канал связи.
